# Getting Started
To get local copy of the project and run
 ``git clone git@github.com:levy002/hello-rails-back-end.git``

## Database Setup
Setup db

1. ``su - postgres``
2. ``psql``
4. ``rails db:setup``
5. ``rails db:create``

# Tests

1. ``bundle exec rspec``
2. ``rubocop``

### Run your tests

1. ``rake``
2. ``rspec``

### To run rubocop we use:

1. `rubocop`

### To autocorrect offenses with rubocop we use:
1. `rubocop --auto-correct-all` or
2. `rubocop -A`

Have fun with TDD!
